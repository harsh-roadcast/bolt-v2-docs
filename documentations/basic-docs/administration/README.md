# Administration

The Administration module is the control center of your Bolt V2 workspace. It provides the tools necessary to manage your organization's hierarchy, secure your data through permissions, and handle financial operations.

Designed for Fleet Managers and System Administrators, this section covers the setup and maintenance of your digital workforce.

#### Key Administrative Capabilities

* User Provisioning: Create, edit, and suspend user accounts instantly.
* Permission Control: Assign pre-built roles (e.g., "Viewer", "Manager") or create custom permission sets.
* Financial Oversight: Monitor daily deductions, view transaction logs, and download invoices.
* Audit Logging: Track "Who did what and when" for security and compliance.
